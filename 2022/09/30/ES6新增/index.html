ES6新增

发表于2022-09-30更新于2022-09-30

变量

新增let、const定义变量

var的缺点：

  1. var定义的变量能够重复声明并且不会报错，容易造成冲突
  2. var定义的变量可以修改，没有原生的常量概念
  3. 不支持块级作用域，var属于函数级作用域
  • const是常量，常量命名要用大写字母，值只能定义一次不可更改
  • let和const的区别是：let是变量可以修改，const是常量不可以修改

let与const的相同点：

  1. 同一个变量名不能重复声明
  2. 支持块级作用域
  3. 存在暂时性死区（变量要先定义才能使用）

解构赋值

通过一个外部包裹的[]或者{}表示对数组或对象内部的元素进行提取或者赋值，简化操作。

解构赋值注意事项：

  1. 变量和数据要一一对应
  2. 等号右侧的数据必须合法
  3. 声明和赋值不呢能够分开

用挺多的就不举例了

函数

箭头函数

可以使用更简短的语法定义一个函数，几乎可以替代传统的函数定义function(){}

  1. 箭头函数没有自己的this，他的this要根据作用域链往外部寻找上一层的this

    在箭头函数出现之前，每一个新函数根据它是被如何调用的来定义这个函数的 this 值：

    • 如果该函数是一个构造函数，this 指针指向一个新的对象
    • 在严格模式下的函数调用下，this 指向undefined
    • 如果该函数是一个对象的方法，则它的 this 指针指向这个对象
    • 等等

  2. 箭头函数没有arguments

  3. 箭头函数没有super

  4. 箭头函数没有new.target

  5. 箭头函数不能用于构造函数（不能new它）

数组

数组方法新增：

  1. map
  2. reduce
  3. filter
  4. forEach

map会遍历每一个数组元素，并且往往用于建立数组元素与某些其他值的联系

reduce会以一个累加器为基础，遍历并处理每一个数组元素之于累加器，reduce内部记得return以将值作用与累加器。往往用于对数组进行处理并希望得到某一个值的场景

filter根据条件进行过滤，return true、false决定一个数组元素的去留

forEach遍历数组，第一参数是item，第二参数是index
