什么是DOM遍历

DOM遍历基本上意味着在组成网页的DOM结点上查找。遍历DOM时，可以将文件树上移到任何所需的元素。还可以将层次结构下移到给定的某一个子节点，或者移动到同级结点。
下面给出的是HTML页面的树（DOM树）。使用JQuery遍历，可以轻松地在向上向下侧向移动。

遍历方法

遍历祖先（向上找）

  • parent()
  • parents()
  • parentsUntil()
parent()方法

调用了parent()方法后，在当前DOM结点上上移一个级别。

语法：

$(selector).parent(filter) // filter（可选） 指定选择器表达式以匹配元素

parents()方法

parents()方法会返回所选元素的所有祖先元素。

语法：

$(selector).parents(filter) // filter（可选）指定选择器表达式以匹配元素

parentUntil()方法

返回两个给定参数之间的所有祖先元素。

语法：

$(selector).parentsUntil(stop, selectorExpression)
参数：

  • stop（可选）
  • selectorExpression（可选）选择器表达式，元素或jQuery对象，以使元素与之匹配 注意：要返回多个祖先，需要用逗号分割每个表达式

遍历后代（向下找）

  • children()
  • find()
children()方法

返回所选的元素的直接子级。

语法：

$(selector).children(selectorExpression)

参数：

  • selectorExpression （可选）指定选择器表达式以匹配元素

find()方法

返回指定参数匹配的所有后代元素。

语法：

$(selector).find(selectorExpression)

参数：

  • selectorExpression （可选）选择器表达式、元素或者jQuery对昂，以将元素与之匹配 注意：要搜索多个后代，请用都还分割每个表达式

遍历兄弟（横向遍历）

  • siblings()
  • next()
  • nextAll()
  • nextUntil()
  • prev()
  • prevAll()
  • prevUntil()
sibling()方法

返回所选元素的所有同级元素

next()方法

返回所选元素之后的一个同级元素

nextAll()方法

返回所选元素之后的所有同级元素

prev()方法

返回所选元素之前的一个同级元素

prevAll()方法

返回所选元素之前的所有同级元素

nextUtil()方法和prevUtil()方法

nextUtil()返回两个给定参数之间的所有之后的同级元素
prevUtil()返回两个给定参数之间的所有先前的同级元素

过滤

在通过遍历方法获取某一些DOM结点的时候，也可以通过过滤方法筛选出各种元素。

  • first()
  • last()
  • eq()
  • filter()
  • not()
  • has()
  • is()
first()方法

返回所选元素（能够选择到一个或多个）的第一个元素（选第一个）

last()方法

返回所选元素的最后一个元素

eq()方法

返回具有选定元素的特定索引号的元素

索引号始终从0开始，因此第一个数字将具有索引0，而不是1

filter()方法

返回与特定条件匹配的元素

此方法过滤所有不符合筛选条件的元素，并返回的那些匹配项

not()方法

返回不符合特定条件的元素

与filter()方法的作用相反

has()方法

返回与指定选择器匹配的所有元素，其中包含一个或者多个元素

is()方法

检查所选元素之一是否与给定参数匹配(选择的元素is给定参数?true:false)

如果这些元素中的至少一个与给定参数匹配，则此方法将返回true，否则返回false

is只会进行一个判断，返回true或者false，不会对某一个DOM结点进行某种操作
